観覧時のお願い

花火大会へご来場頂くお客様へ新型コロナウイルス対策のご協力をお願い致します。

下記該当のお客様はご入場出来ません。
  • 来場当日検温し、発熱が37.5℃以上あった方。
  • 大会当日に体調が優れない方。 ( 咳、呼吸困難、強い倦怠感、味覚・嗅覚障害、下痢、嘔気・嘔吐等の症状 )
  • 新型コロナウイルス感染症の陽性判定を受けている方、自宅待機指示を受けている方。
  • 過去14日以内に新型コロナウイルス感染症の陽性と判明した方との濃厚接触がある方。
  • 過去14日以内に同居家族や身近な知人のコロナ感染が疑われる方。

Program
